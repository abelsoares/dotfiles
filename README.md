# dotfiles

A fork of the awesome [ruimarinho/dotfiles](https://github.com/ruimarinho/dotfiles) tailored to my own personal preferences.  

Checkout the original [README.md](https://github.com/ruimarinho/dotfiles) for requirements and instalation instructions.  

This fork does not provide any enhancement regarding the robustness of setup and personal customization, if you still want to use this fork, these are some guidelines to personalization: 

- Update `macos/default` to update your personal macOS preferences. 
- Update `iterm/profiles/default.json` to customize your `iterm` defaults. 
- Update `brew/Brewfile` to reflect your own brew packages installation.
